# 음악스트리밍


## 소개
---
평소 음악을 좋아하여, 개발공부 초기에 음악을 듣고, 자신만의 음악리스트를 만들고, 음악에 관해 이야기할 수 있는 음악 스트리밍 사이트를 만들어봤다.

회원가입(이메일 인증),게시판(페이징,댓글--등록/수정/삭제,다중 이미지 업로드),인피니티 스크롤(ajax),음악재생,음악 즐겨찾기,음악 좋아요(ajax - 실시간 반영),DJ 랭킹(크롤링)등등의 기능 구현했다.
